# Project Overview

## 프로젝트명
LLM 모델을 활용한 프롬프트 기반 POI 추천 알고리즘 개발

## 프로젝트 목표
한국어 자연어 프롬프트를 이해하고, 장소 의미 유사도와 위치 정보를 결합해 추천 가능한 POI 후보를 제시하는 하이브리드 추천 파이프라인을 구현한다.

## 문제의식
일반적인 생성형 QA는 장소를 텍스트로만 설명하는 데 그치기 쉽다.
이 프로젝트는 장소 설명, 유사 장소 검색, 좌표 변환, 지도 시각화를 연결하여 실제 추천 경험에 가까운 결과를 만들고자 했다.

## 구현 포인트
- 한국어 LLM 기반 질의응답
- KoBERT 기반 장소명 임베딩 및 의미 유사도 검색
- Kakao API 기반 주소 → 좌표 변환
- Folium 기반 지도/클러스터/경로 시각화

## 기대 효과
- 단순 텍스트 질의응답을 넘어서는 POI 추천 경험 제공
- NLP + 위치 정보 + 시각화의 결합 구조 검증
- 향후 personalized ranking, API 서버화, 데모 UI로 확장 가능한 기반 마련
